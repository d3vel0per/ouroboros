# Code of Conduct

## Our Pledge

We pledge to make participation in our community a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

### Positive behavior includes

- Being respectful and inclusive
- Gracefully accepting constructive criticism
- Focusing on what is best for the community
- Showing empathy towards other community members

### Unacceptable behavior includes

- Harassment, trolling, or derogatory comments
- Personal or political attacks
- Public or private harassment
- Publishing private information without permission
- Any other conduct which could reasonably be considered inappropriate

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that are not aligned with this Code of Conduct.

## Reporting

If possible, contact maintainers through a private path for conduct-sensitive reports.

If a private path is not available, open a GitHub issue with the `conduct` label and include only the minimum information needed to route the report. Do **not** include private personal details, doxxing content, or material that would further harm the affected person.

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, code review, and related community spaces associated with Ouroboros.
